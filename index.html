Overview

Launch your own immersive VR meditation app in days, not months.

The VR Meditation App Template is a fully production-ready, highly optimized Unity3D project designed for creators, wellness centers, therapists, and businesses who want to launch their own VR meditation application on Meta Quest 3.

You no longer need to spend months coding the video player, UI logic, localization systems, or configuration engines from scratch. With this template, you or your developer simply drag-and-drop your content, configure a single file, and build a premium-grade application for Meta Quest 3, Quest 2, and other VR platforms.

Zero-to-low code

If you can edit a text file and move media into folders, you can build your app. No complex C#.

Built for Quest 3

Optimized for high-resolution 360° playback, spatial comfort and seamless controller tracking.

100% white-label

Replace logos, UI assets and colors to fully match your brand identity.

Flexible progression

Let users explore all meditations freely, or unlock them step-by-step.

Key features & capabilities

Everything a premium meditation app needs — already built.

🎥

High-performance 360° video player

  • Optimized for immersive equirectangular projections
  • Up to 8K on Meta Quest 3 (codec dependent)
  • Comfortable fade-in/out to prevent motion discomfort
🎧

Dual-source audio engine

  • Ambient track — loopable nature sounds, binaural beats, pads
  • Voiceover guide synced precisely with the visuals
  • Independent volume for both sources
⚙️

Config-driven architecture

  • One human-readable JSON file controls everything
  • Define videos, audio, timing, titles & durations
  • Set unlock prerequisites per session
🏆

Progress tracking & history

  • Visual calendar / history of completed sessions
  • Total meditation time & streak statistics
  • Saved locally on device — no backend required
🌍

Out-of-the-box multi-language support

  • Robust localization system — add JSON language files and every UI element, description and button adapts automatically to the user's device language.
How it works

The 3-step "drop-and-build" setup.

Customize and launch faster than any custom development cycle.

1

Add your content

Drop your media into dedicated Unity folders.

/Videos360/ 360° MP4s
/Audio/ ambient + vocal guides
/Thumbnails/ menu card images
2

Edit the config

Open meditation_config.json and define each session — videos, audio, timelines and unlock rules.

3

Build & launch

Set Android as your Unity target, hit Build, and your custom APK is ready for sideloading or the Meta Quest Horizon Store (App Lab).

meditation_config.json
{
  "progressionMode": "linear", // "free" (all unlocked) or "linear"
  "sessions": [
    {
      "id": "beach_morning",
      "title_key": "SESSION_BEACH_TITLE",
      "thumbnail": "thumb_beach",
      "video_filename": "beach_360_4k.mp4",
      "ambient_audio": "waves_loop.wav",
      "guide_audio": "breathing_guide_eng.wav",
      "guide_start_delay_seconds": 5.0,
      "unlocked_by_default": true
    },
    {
      "id": "forest_night",
      "video_filename": "forest_360_4k.mp4",
      "ambient_audio": "forest_crickets.wav",
      "guide_start_delay_seconds": 10.0,
      "unlocked_by_default": false
    }
  ]
}
